# Icon Requirements

## Tray Icon
- File: `tray-icon.png`
- Size: 16x16 or 32x32 pixels (PNG format)
- Should be a simple icon that works well at small sizes
- Place in the `electron-app/` directory

## PWA Icons
- Files: `public/icon-192.png` and `public/icon-512.png`
- Sizes: 192x192 and 512x512 pixels (PNG format)
- Used for PWA installation on mobile devices
- Place in the `electron-app/public/` directory

You can convert the SVG logo from `svg/logo.svg` to these PNG formats using any image editor or online converter.
